windows磁盘映射

磁盘映射就是将本地某个文件夹或者局域网中的某个计算机的某个目录映射成本地驱动器号，就是说把本地的文件夹或者网络上其他机器的共享的文件夹映射成为自己机器上的一个磁盘，这样可以可以更方便的打开相应的文档，下面作者分享几个在windows下比较实用的磁盘映射命令。

进行下面的之前请首先按组合键win+R打开运行对话框，输入cmd，打开windows下的dos窗口，这里主要用到了 subst 和 net use 两个命令，对这两个命令不太懂的可以通过在 dos 窗口下输入 subst /? 和 net use /? 命令来获取帮助信息。

1．映射虚拟磁盘subst命令（这个命令可以简化很多操作，比如我们把一个文件放在一个路径很深的文件夹中，每次我们想要操作这些文件就必须沿着文件夹的路径一级一级地打开，这样操作很费时。如果使用虚拟硬盘就可以很方便的进入到我们需要操作的文件夹了）

（1）利用subst命令可以将本地的一个文件夹映射成为本地的一个硬盘，例如下面将D:\documents这个目录映射成为本地磁盘M盘：

subst M: D:\documents

这样我们打开资源管理器的时候，就会发现现在多了一个M盘。

（2）删除已经映射的虚拟磁盘，例如这里要删除刚才映射的M盘：

subst M: /D

（3）直接输入subst命令可以察看当前的虚拟磁盘的情况，命令如下：

subst

2．net use 命令映射一个网络驱动器

我们在使用局域网是经常要映射很多的网络驱动器，但是这些网络驱动器不会自动删除，在每次开机时都会要我们输入密码。我觉得这很繁琐，能不能简化一下操作呢？

其实是可以的，那就是使用net use命令，使用这个命令就可以自动映射一个网络驱动器。我们如果将这些命令保存在一个批处理文件中，然后让在启动文件夹中插入一个快捷方式，运行这个批处理文件，那么我们就不需要每次开机时输入密码了，不过对于windows 2000 professional中每次开机时还会提示输入密码，不过此时我们可以选择按以下Esc键，然后让我们的批处理文件来进行映射操作。对于2000 Server好像在每次开机时如果是用net use 命令映射的驱动器在第二次开机时不会自动连接。

下面通过几个实例介绍一下net use 命令的使用方法

（1）假定有一台机器名为phpernote，在这台机器上有一个共享文件夹，它的共享名称为Share，而且可以使用用户名user登录，user的密码为123456，那么将这个网络文件夹映射为本地的H:盘的命令如下：

net use H: \\phpernote\Share 123456 /USER:user

（2）要删除网络驱动器H:盘，可以使用下面的命令：

net use H: /D

（3）直接使用net use命令可以察看已经映射的网络驱动器，命令如下：

net use
